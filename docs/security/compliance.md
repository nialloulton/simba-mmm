# Compliance — GDPR, Data Protection Act, and Cyber Essentials

Simba maintains compliance with key privacy regulations and security certifications to ensure your data is handled responsibly and legally.

## Cyber Essentials Certification

Simba is **Cyber Essentials certified**, a UK Government-backed scheme that demonstrates adherence to fundamental cybersecurity standards:

- Boundary firewalls and internet gateways
- Secure configuration
- Access control
- Malware protection
- Patch management

Cyber Essentials certification is required for UK Government contracts and recognized as a baseline security standard across industries.

## GDPR Compliance

Simba is fully compliant with the **General Data Protection Regulation (GDPR)**:

### Data Minimization
- We collect only the data necessary to provide the service
- Account creation requires only email address and password
- No unnecessary personal data is collected or stored

### Zero-Retention Logging
- Strict logging policies ensure that personal data is not retained in system logs beyond what is necessary for service operation

### User Rights
Under GDPR, you have the right to:

| Right | Description |
|-------|-------------|
| **Access** | Request a copy of your personal data |
| **Rectification** | Correct inaccurate personal data |
| **Erasure** | Request deletion of your personal data ("right to be forgotten") |
| **Restriction** | Restrict processing of your personal data |
| **Portability** | Receive your data in a portable format |
| **Objection** | Object to certain types of processing |

To exercise any of these rights, contact us at **info@pymc-labs.com**.

## Data Protection Act

Simba complies with the UK **Data Protection Act 2018**, which supplements GDPR in UK law.
