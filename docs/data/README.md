## Quick Checklist

Before uploading data to Simba, ensure you have:

- [ ] A target KPI time series (e.g., weekly revenue, conversions, or sales)
- [ ] Media spend or activity data by channel (e.g., impressions, GRPs, spend)
- [ ] At least 1–2 years of historical data (more is better)
- [ ] Consistent time granularity across all variables (weekly recommended)
- [ ] No large unexplained gaps in the data
