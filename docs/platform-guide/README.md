# Platform Guide

This guide covers every major feature of the Simba MMM platform, organized around the four-step workflow: **Audit, Measure, Predict, Optimize**. Each page explains what a feature does, how to use it, and how to interpret the results.

## The Simba Workflow

### Step 1: Audit

- **[AI Data Auditor](./data-auditor.md)** --- Automated validation and quality scoring for every data source. Detects anomalies, handles missing values, checks schema integrity, and produces a data health score before modeling begins.

### Step 2: Measure

- **[Incremental Measurement](./measurement.md)** --- Causal attribution across all marketing channels. Separates base sales from media lift, integrates lift tests, and isolates the true incremental value of each channel.

### Step 3: Predict

- **[Scenario Planning](./scenario-planning.md)** --- Multi-scenario forecasting with aggressive, base, and conservative projections. Includes uncertainty bands, what-if budget reallocation, and revenue forecast charts for comparing possible futures.

### Step 4: Optimize

- **[Budget Optimization](./budget-optimization.md)** --- Risk-adjusted spend allocation with multi-week carryover awareness. Delivers precise per-channel recommendations that balance ROI targets with risk appetite.

## Model Setup and Configuration

- **[Model Configuration](./model-configuration.md)** --- No-code interface for setting priors, saturation curves, and decay parameters on a per-channel basis. Covers distribution selection, example configurations, and when to override defaults.

- **[Smart Defaults](./smart-defaults.md)** --- Auto-generated starting points derived from historical data and industry benchmarks. Explains when defaults are sufficient and when domain expertise should override them.

- **[Long-Term Effects](./long-term-effects.md)** --- Modeling lasting brand impact beyond immediate performance. Covers memory decay curves, behavioral response, and seasonal patterns. Available on Pro tier and above.
